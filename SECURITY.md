# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in EIDARA, please report it responsibly.

**Email:** [javier@eidara.dev](mailto:javier@eidara.dev)

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

I'll acknowledge your report within 48 hours and provide an update within 7 days.

## Scope

EIDARA runs entirely on your local machine. There are no servers, APIs, or cloud components. Security concerns typically involve:

- The compiler (`compile.py`, `validators.py`, `watcher.py`)
- The constitution and governance rules
- The INSTALL guide and setup process

## Out of Scope

- Your AI provider's security (Claude, GPT, etc.)
- Your operating system or Python installation
- Personal data you store in your own vault

## Disclosure

I believe in responsible disclosure. I'll work with you to understand and fix the issue before any public discussion.
