Philosophy

No other open-source memory system publishes stress test results across multiple LLM models. We do because the system was designed to work with ANY model — and we need to prove it actually does.

What the 7 blocks test

  • Block 1 Reading comprehension — can the AI find specific info in BRAIN.md?
  • Block 2 Writing protocol — does it follow W1-W15 correctly?
  • Block 3 Edge cases — malformed files, Unicode, empty entries
  • Block 4 Recovery — can it follow RECOVERY.md procedures?
  • Block 5 Consensus — does it respect 3/3 flags?
  • Block 6 Protected files — does it refuse to edit checksummed files?
  • Block 7 Cross-platform — does it work in Light mode?
